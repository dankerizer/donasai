A simple, lightweight donation plugin with manual bank transfer support and donor dashboard.

== Description ==

wp-donasi is a donation plugin designed for Indonesian nonprofits, mosques, and communities. It focuses on simplicity and manual bank transfer support.

**Key Features (Free):**
*   **Campaign Management**: Create unlimited fundraising campaigns.
*   **Manual Payment**: Support for offline bank transfers (BCA, Mandiri, etc.).
*   **Donation Form**: Mobile-friendly form with preset amounts.
*   **Admin Dashboard**: React-powered dashboard to track donations.
*   **Donor History**: Shortcode `[wpd_my_donations]` for donors to view their history.
*   **CSV Export**: Export donation data for reporting.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/wp-donasi` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to **Donasi > Settings** to configure your Bank Account details.
4.  Create a new Campaign via **Donasi > Campaigns**.

== Frequently Asked Questions ==

= How do I display the donation form? =
The donation form is automatically displayed on the Campaign single page. You can also use shortcodes (coming in Pro).

= How do donors confirm payment? =
Donors receive an email instruction. Admins can manually mark donations as "Complete" in the Dashboard.
