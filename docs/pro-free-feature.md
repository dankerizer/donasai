# donasai: Free vs Pro Features

Berikut adalah perbandingan fitur antara **donasai Free** (tersedia di WordPress.org) dan **donasai Pro** (Add-on berbayar).

## Tabel Perbandingan Singkat

| Fitur | Free | Pro |
| :--- | :--- | :--- |
| **Campaign Management** | Unlimited Campaign Basic | Unlimited + Advanced (Kategori Khusus, Lokasi, Layout Builder) |
| **Donation Form** | Preset Amount, Custom Amount | **Recurring Donation** (Bulanan/Tahunan), Fee Coverage |
| **Payment Gateway** | Manual Transfer + 1 Gateway Global | **Gateway Lokal** (Midtrans, Xendit, Mozu), E-Wallet |
| **Reporting** | Dashboard & Grafik Dasar | **Advanced Reporting**, Export Detail CSV/PDF |
| **Donatur** | List Donatur, Email Konfirmasi | **Donor Dashboard**, Download Receipt PDF, WhatsApp Notif |
| **Integrasi** | Shortcode Standar | Webhook, Pixel Tracking (FB/TikTok), CRM Integration |

---

## Detail Fitur

### 1. Campaign Management
*   **Free:**
    *   Membuat campaign donasi tidak terbatas.
    *   Set target donasi, deadline, dan progress bar.
    *   Tampilan landing page standar yang responsif.
    *   Kategori dasar (Zakat, Infak, Wakaf, Umum).
*   **Pro:**
    *   **Advanced Layouts:** Pilihan template landing page premium.
    *   **Custom Fields:** Menambahkan field khusus pada campaign (lokasi proyek, penanggung jawab).
    *   **Urgency Tools:** Countdown timer yang lebih kustomizable.

### 2. Donation Form & Flow
*   **Free:**
    *   Form donasi tunggal per campaign.
    *   Pilihan nominal donasi (preset & custom).
    *   Input data donatur standar (Nama, Email, No HP, Pesan).
    *   Opsi donasi sebagai anonim (Hamba Allah).
*   **Pro:**
    *   **Recurring Donations:** Opsi donasi rutin (subscription) bulanan atau tahunan.
    *   **Fee Coverage:** Opsi bagi donatur untuk menanggung biaya admin/gateway.
    *   **Multi-step Form:** Form donasi bertahap untuk UX yang lebih baik pada data kompleks.
    *   **Conditional Logic:** Tampilkan field tertentu berdasarkan pilihan donatur.

### 3. Payment Gateway
*   **Free:**
    *   **Offline Transfer:** Instruksi transfer bank manual & konfirmasi admin.
    *   **Basic Gateway:** Integrasi standar (contoh: PayPal atau Stripe Basic).
*   **Pro:**
    *   **Local Payment Giants (Indonesia):** Integrasi native dengan Midtrans, Xendit, Tripay, iPaymu.
    *   **QRIS & E-Wallet:** Support pembayaran instan via GoPay, OVO, ShopeePay (melalui gateway).
    *   **Virtual Accounts:** Automatisasi pengecekan mutasi bank via VA.

### 4. Donor Management & Dashboard
*   **Free:**
    *   Halaman "My Donations" sederhana (list riwayat donasi user).
    *   Email notifikasi sukses donasi standar.
*   **Pro:**
    *   **Advanced Donor Dashboard:** User bisa mengelola langganan donasi rutin (pause/cancel).
    *   **PDF Receipts:** Download kuitansi/invoice donasi otomatis dalam format PDF.
    *   **WhatsApp Notifications:** Kirim notifikasi status donasi via WhatsApp (perlu gateway WA).

### 5. Reporting & Analytics
*   **Free:**
    *   Dashboard admin dengan ringkasan total donasi & jumlah donatur.
    *   Grafik tren donasi harian/mingguan sederhana.
    *   Export data donasi sederhana.
*   **Pro:**
    *   **Deep Analytics:** Analisa performa per campaign, konversi, dan retensi donatur.
    *   **Comparative Reports:** Bandingkan performa antar campaign atau periode waktu.
    *   **Advanced Export:** Export data custom dengan filter mendalam.

### 6. Marketing & Integrasi
*   **Free:**
    *   SEO basic untuk halaman campaign.
*   **Pro:**
    *   **Pixel Tracking:** Integrasi mudah dengan Facebook Pixel, TikTok Pixel, Google Analytics 4.
    *   **Fundraiser/Affiliate:** Sistem bagi user untuk menjadi fundraiser dan menyebarkan link referral unik.
    *   **Webhooks:** Kirim data donasi ke aplikasi pihak ketiga (Zapier, Google Sheets, CRM).

---
*Catatan: Fitur Pro dapat dibeli secara terpisah atau dalam satu bundle lengkap.*
